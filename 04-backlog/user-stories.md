Epic: Digital Loan Application


US-01: Eligibility Check
As a customer,
I want to check my loan eligibility before applying,
So I can decide whether to proceed.

Acceptance Criteria:
- Eligible loan range is displayed
- Estimated monthly payment is shown
- Clear next-step call to action

Priority: High


US-02: Digital Loan Application
As a customer,
I want to submit my loan application digitally,
So I can apply without visiting a branch.

Acceptance Criteria:
- Personal and financial information form
- Mandatory fields validation
- Progress indicator visible

Priority: High


### US-03: Document Upload
As a customer,
I want to upload required documents online,
So I can complete my application remotely.

Acceptance Criteria:
- Supports PDF and JPG formats
- File size limit defined
- Upload success confirmation shown

Priority: High


### US-04: Application Status Tracking
As a customer,
I want to track the status of my loan application,
So I know what stage it is in.

Acceptance Criteria:
- Status labels (Submitted, Under Review, Approved, Rejected)
- Estimated processing time displayed
- Notification on status change

Priority: Medium
